my projects

To see the deployed version (Live) click in the globe icon. To see the source code, in the Github icon.

Github User Stats

React app that provides several stats and general data about a user's github.

It uses GitHub's API for fetching the data, React Router for navigation, Auth0 for login, Context API to avoid props drilling and Fusion Charts to show the different graphs.

YelpCamp

Application using templating: EJS to implement JS in HTML. Bootstrap 5. Mongo (cloud) as DB, ExpressJS to help with the middleware and backend.

JOI schema validations. Passport library for authentication. Multer and Cloudinary for Image Upload. Mapbox for working with GeoJSON and adding interactive maps.

Cocktails DB

React app that provides Cocktails data using CocktailDB API. It also uses React Router for navigation, Context API, Fetch for fetching the data and finally UseState, UseContext and UseEffect Hooks.

Stock Photos

React app that provides pictures from Unsplash according to a search term. It uses Unsplash's API for fetching the data, UseState and UseEffect Hooks. It also uses conditional rendering, DOM manipulation and controlled inputs.
